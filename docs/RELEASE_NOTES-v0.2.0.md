# PlexusX 0.2.0

Take control of how your games and desktop look — vibrance, brightness, gamma and eye
care, all in one place.

## Features

- **Digital Vibrance 0–200%** — push saturation past the NVIDIA driver's 100% ceiling,
  system-wide
- **Brightness calibration** and **Gamma** sliders for dialling in your display
- **Eye care** — blue-light reduction (warm light) for late-night sessions
- **Games Hub** — finds the supported games installed on your PC and tunes them.
  **Rust** supported: graphics quality, FPS limit, effect toggles, plus Launch and
  Verify/Repair. Your config is backed up automatically before anything is changed.
- **Presets** — Natural / Standard / Vivid / Max in one click
- **Dark and light (black & white) themes**, with an animated plexus background

## Install

Download **`PlexusX-Setup-0.2.0.exe`** below and run it. It installs in seconds and adds a
Start Menu shortcut (a desktop shortcut is optional).

Open PlexusX with **Ctrl+Alt+V**, from the tray icon, or from the Start Menu.

## Requirements

- Windows 10 or 11
- An **NVIDIA GPU** (required for the 0–100% driver vibrance)

## Note on the Windows warning

PlexusX isn't code-signed yet, so Windows may show **"Windows protected your PC"**.
Click **More info → Run anyway**. This is normal for independent apps and will go away
once the app is signed.
